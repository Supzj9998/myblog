Python是由荷兰人吉多·范罗苏姆(Guido van Rossum)在 1989 年在圣诞节期间创建、1991 年发布的高级编程语言，以简洁易读著称。
越是接近人类的语言就越高级，python就是高级编程语言。相反，越接近机器语言的就越低级，例如汇编

Python在机器学习领域提供了Pytorch,TensorFlow等方便的库供我们使用，这也是笔者学习Python的主要原因

# 一、基础知识

## 1.输入输出

输出函数 print()

```python
print() #括号中填需要输出的内容
```

输入函数 input()

```python
input() #括号中填需要输入的内容
```

## 2.注释

单行注释 #

```python
# 这是一个单行注释
```

多行注释 """ """或''' '''

```python
"""
  这是一个
  多行注释
"""
'''
  这也是一个
  多行注释
'''
```
